# SHOPCART - E-commerce Website

A static e-commerce website built with HTML, CSS, and JavaScript, featuring product browsing, shopping cart functionality, and a responsive design.

## Description

SHOPLANE is an online store that offers clothing and accessories for men and women. The website provides a seamless shopping experience with a clean, modern interface. Users can browse products, add items to their cart, and proceed to checkout.

## Features

- **Product Browsing**: Display of clothing and accessories categorized by gender
- **Shopping Cart**: Add, remove, and manage items in the cart
- **Search Functionality**: Search for specific clothing and accessories
- **Responsive Design**: Optimized for various screen sizes
- **Image Slider**: Featured product carousel on the homepage
- **Checkout Process**: Complete checkout flow with order placement

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and layout
- **JavaScript**: Dynamic functionality
- **jQuery**: DOM manipulation and AJAX requests
- **Slick Carousel**: Image slider implementation
- **Font Awesome**: Icons
- **Google Fonts**: Typography (Lato and Source Sans Pro)

## Installation and Usage

1. Clone or download the repository
2. Open `index.html` in a web browser
3. Navigate through the website using the header links
4. Browse products and add items to cart
5. Proceed to checkout when ready

No server setup is required as this is a static website.

## Pages

- **Home (index.html)**: Main landing page with slider and featured content
- **Clothing**: Browse men's and women's clothing
- **Accessories**: Browse men's and women's accessories
- **Cart**: View and manage shopping cart items
- **Checkout**: Complete purchase process
